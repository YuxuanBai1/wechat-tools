### 使用本项目前，请您先了解本项目的 ~~不足~~ 特性：

1. 本项目不支持自动寻找用户微信所在路径；

2. 本项目对用户电脑的分辨率与缩放有要求；

第二条您看完实现原理后就明白怎么回事了

### 实现原理

本项目代码是通过系统api获取微信文字并模拟用户点击、输入等实现操作。所以缩放率及分辨率最优为125%和1920 X 1080
